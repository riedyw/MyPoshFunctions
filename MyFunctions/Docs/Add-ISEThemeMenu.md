# Add-ISEThemeMenu

## SYNOPSIS
Adds a theme to the ISE Theme menu

## SYNTAX

```
Add-ISEThemeMenu [[-MenuName] <String>] [[-ShortcutDirectory] <String>] [<CommonParameters>]
```

## DESCRIPTION
Adds a theme to the ISE Theme menu

## EXAMPLES

### EXAMPLE 1
```
Add-ISEThemeMenu
```
