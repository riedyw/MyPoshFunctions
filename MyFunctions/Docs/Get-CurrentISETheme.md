# Get-CurrentISETheme

## SYNOPSIS
Gets current ISE theme

## SYNTAX

```
Get-CurrentISETheme
```

## DESCRIPTION
Gets current ISE theme.
Hex colors are converted to ARGB
and added back to the returned objects.

## EXAMPLES

### EXAMPLE 1
```
$CurrentISETheme = Get-CurrentISETheme
```

Assigns Current ISE theme properties to CurrentISETheme
