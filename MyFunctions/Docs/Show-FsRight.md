# Show-FsRight

## SYNOPSIS
To list all potential file system rights

## SYNTAX

```
Show-FsRight [<CommonParameters>]
```

## DESCRIPTION
To list all potential file system rights and to return a psobject with the properties Name, Type, Hex, Dec

## EXAMPLES

### EXAMPLE 1
```
Show-FsRight
```
