# Remove-ISETheme

## SYNOPSIS
Deletes an ISE theme from the ISE.

## SYNTAX

```
Remove-ISETheme [[-ThemeName] <String>] [<CommonParameters>]
```

## DESCRIPTION
Deletes an ISE theme from the ISE.

## EXAMPLES

### EXAMPLE 1
```
Remove-ISETheme "Monokai"
```

Deletes an ISE theme from the ISE
